Solidis

A browser-based 3D WebGL library

What is Solidis?

sol·id

noun /ˈsäləd/

: having three dimensions, as a geometrical body or figure.

Solidis is a browser-based 3D WebGL library designed to power your games and applications.

Examples

After cloning this repository, run the following commands:

npm install
gulp production

Navigate to the examples directory in your browser and you should see different shapes being rendered.

Documentation

The documentation in the docs directory is generated by jsDoc. Documentation is automatically generated when running the gulp production step. Make sure to dump the docs folder into the git@github.com:codeboxsystems/solidis-docs repo when you're ready to deploy. Head over to that repo for the exact deploy steps.

JSDoc CSS Styles

Make sure to move site.codeboxsystems.css into solidis/node_modules/gulp-jsdoc/node_modules/ink-docstrap/template/static/styles/site.codeboxsystems.css

Testing

Units tests are locations in the tests directory and run using Karma.

webgl-utils.js

This file contains functions every webgl program will need a version of one way or another. Instead of setting up a context manually it is recommended to use. This will check for success or failure. On failure it will attempt to present an approriate message to the user. gl = WebGLUtils.setupWebGL(canvas); For animated WebGL apps use of setTimeout or setInterval are discouraged. It is recommended you structure your rendering loop like this. function render() { window.requestAnimationFrame(render, canvas); // do rendering ... } render(); This will call your rendering function up to the refresh rate of your display but will stop rendering if your app is not visible.
